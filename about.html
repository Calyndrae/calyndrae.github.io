Profile // Developer

Behind the Code.

作为一名全栈开发者与系统研究者，我热衷于拆解复杂的系统。从深入 C++ 的内存快照，到设计红蓝对抗中的隐蔽驻留链路，再到 Minecraft 数据包的底层状态机重构——我享受在底层逻辑中寻找最优解的过程。

Philosophy

我相信，最顶级的工程应当像艺术品一样优雅。代码不应当是条件语句的无脑堆砌，而是数据流的完美分发。无论是管理 Windows Server 下的 PM2 守护进程，还是利用 1.21 组件语法开发无延迟的沙盒游戏模式，"高内聚、低耦合" 始终是我架构设计的核心。

Journey & Evolution

2026 - Present

企业级渗透防御演练
